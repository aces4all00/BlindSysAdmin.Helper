# Test-IsWindows

## SYNOPSIS
Tests if PowerShell is running on Windows

## SYNTAX

```
Test-IsWindows
```

## DESCRIPTION
Tests if PowerShell is running on Windows.
Helps cross-platform scripts identify if they are running on Windows.

## EXAMPLES

### EXAMPLE 1
```powershell
Test-IsWindows
```
```output
TRUE
```

Test if PowerShell is running on a Windows OS
